# 思政案例库通用编写模板
## ——基于"立德树人·课程思政·思政课程·大思政课"框架

> **使用说明**：本模板面向任意素材的思政案例开发。无论素材来自科学领域、历史事件、日常现象、文艺作品，还是正面典型或反面案例，均可按此模板完成从"原始素材"到"可用案例"的标准化转化。

---

## 二、思政元素提取工作表

> 对素材进行系统性"思政挖掘"，每个有效框架可独立用于不同场景。

### 2.1 "四层价值坐标系"扫描

> 逐层扫描素材，标记有潜力的思政元素（打√），并简要说明联结逻辑。

| 层次 | 核心问题 | 素材相关性 | 可提炼的思政点 |
|------|----------|------------|----------------|
| **个人层** | 此素材中的人/事体现了哪种个体品格或精神？ | □有  □无 | |
| **社会层** | 此素材涉及制度、文化、社会关系的哪个面向？ | □有  □无 | |
| **国家层** | 此素材与国家发展、主权、安全、民族复兴有何关联？ | □有  □无 | |
| **人类层** | 此素材是否涉及人类命运、生态文明、文明互鉴？ | □有  □无 | |

**扫描结论**：本素材最强的思政切入点在_____层，具体为：________________________

---

### 2.2 六大核心主题对照表

> 依据《高等学校课程思政建设指导纲要》（教高〔2020〕3号），勾选素材涉及的主题，并写出具体联结点。

| 核心主题 | 政策内涵要点 | 本素材的联结点 | 强度评级 |
|----------|-------------|---------------|---------|
| **政治认同** | 坚定四个自信、维护两个确立、理解中国道路逻辑 | | □强 □中 □弱 □无 |
| **家国情怀** | 爱国主义、天下兴亡匹夫有责、个人命运与国家命运相连 | | □强 □中 □弱 □无 |
| **文化素养** | 中华优秀传统文化、革命文化、社会主义先进文化、文化自信 | | □强 □中 □弱 □无 |
| **宪法法治意识** | 依法治国、尊法守法、法律思维、专业相关法规伦理 | | □强 □中 □弱 □无 |
| **道德修养** | 社会主义核心价值观、职业道德、个人品德、社会公德 | | □强 □中 □弱 □无 |
| **劳动与奋斗精神** | 劳动光荣、工匠精神、创新创业、艰苦奋斗 | | □强 □中 □弱 □无 |

**主题确定**：本案例主打主题_____（1-2个），辅助主题_____（0-1个）。

---

### 2.3 素材深挖三层法

> 从表层现象向深层价值递进挖掘，避免停在表面。

**表层（What）——事实描述层**：
> 素材呈现了什么？时间、地点、人物、事件概要（2-3句话）

________________________________

**中层（Why/How）——行为逻辑层**：
> 为什么会发生这件事？当事人/历史背景/社会结构提供了什么解释？（3-5句话）

________________________________

**深层（What it means）——价值根源层**：
> 这背后体现了什么样的世界观、价值观、历史观？与哪类思政命题产生共鸣？（3-5句话）

________________________________

## 三、案例正文编写模板

> 根据素材类型和应用场景，选择以下四种正文结构之一。

---

### 【模板A】标准叙事型
> 适用于：人物故事、历史事件、新闻事件类素材

---

**案例标题**：________________________（主副标题，总字数10-20字为宜）

**适用课程/场景**：________________________（课程名+具体章节/知识点）

**建议教学时长**：□ 5分钟（课堂导入）  □ 10-15分钟（分析讨论）  □ 30分钟以上（专题研讨）

---

**【情境还原】**（200-400字，客观第三人称，不做价值评判）

________________________________

> *写作提示*：包含时间、地点、人物、关键事件。保留历史或事件的不确定性感，不要提前剧透结果。

---

**【冲突与抉择】**（150-300字，聚焦价值张力）

________________________________

> *写作提示*：呈现当事人面临的两难、困境或关键抉择。这是制造"认知张力"的核心环节——让学生感到这个选择并不容易，从而产生代入感和思考动力。

---

**【行动与结果】**（150-300字）

________________________________

> *写作提示*：描述实际发生了什么，产生了什么历史/社会影响。正面案例强调选择过程的艰难；负面案例客观呈现后果，并注意呈现"制度回应"或"纠错过程"，避免虚无主义。

---

**【思政映射】**（200-400字，教师视角，条目化呈现）

本案例的核心思政价值点：

1. **[价值点名称]**：____________（说明此素材如何生发这一价值点，联结课程知识）
2. **[价值点名称]**：____________（同上）
3. **[价值点名称]**（可选）：____________

---

**【讨论问题设计】**

> 设计2-4个梯度问题，从事实理解到价值内化递进。

- *事实层*（确保理解）：_____________________？
- *分析层*（促进思考）：_____________________？
- *价值层*（引发内化）：_____________________？
- *行动层*（可选）：如果是你，________________？

---

> 描述具体的科学发现、技术突破或工程成就，保持客观专业语言。

________________________________

**【思政价值点提炼】**（条目化）

> 负面案例的思政价值在于：引发学生对正确价值的主动认同，而非被动灌输。

1. _______________（如：本案例说明为何需要____法律/规范）
2. _______________（如：从制度回应中可以看到____的价值）
3. _______________（可选）

---

**【讨论问题设计】**

- *理解层*：这件事为什么会发生？
- *分析层*：如果你是[某角色]，你会如何应对？
- *反思层*：这件事说明什么样的制度或价值观是必要的？
- *行动层*：在你未来的专业实践中，如何避免类似问题？

> *安全提示*：使用负面案例时须在课堂中明确说明"本讨论不是为了批判某人/某群体，而是为了理解某类问题的成因和应对"，建立讨论的安全感。

## 五、多场景应用适配指南

### 场景一：思政课程（思政课堂直接使用）

**核心要求**：与思政课教学目标精准对应，案例须有力支撑理论观点，而非替代理论分析。

**适配操作**：
- 在"思政映射"部分，明确标注对应思政课课程的具体知识点（如"对应《思想道德与法治》第三章第二节"）
- 讨论问题须引导学生运用马克思主义立场、观点、方法分析，而非停留于感性描述
- 每个案例须说明在课程中是作为"导入案例""论证案例"还是"延展案例"使用

---

### 场景二：课程思政（专业课程中融入）

**核心要求**：专业知识与思政价值"有机融合"，不可"两张皮"。融入必须发生在专业知识讲授过程中，而非附加在前后。

**适配操作**：
- 在案例信息卡中必须填写"对应专业知识锚点"
- 标明融入时机（在讲到哪个概念/公式/案例时切入）
- 专业讨论问题与思政讨论问题须有内在关联，而非两个独立问题并排

**各学科适配重点**：

| 学科类型 | 思政融入重点 | 优先使用模板 |
|----------|-------------|-------------|
| 理工科 | 科学家精神、工程伦理、科技与国家安全 | 模板B |
| 人文社科 | 历史唯物主义、文化自信、批判性思维 | 模板A + C |
| 医学 | 医德医风、生命伦理、大爱精神 | 模板A + D |
| 经管法 | 职业道德、法治意识、社会责任 | 模板C + D |
| 艺术体育 | 中华美学精神、民族文化认同、体育精神 | 模板A |
| 农学 | 三农情怀、生态文明、粮食安全 | 模板B + C |

---

### 场景三：大思政课（社会大课堂联动）

**核心要求**：打通课堂内外，将社会现实资源转化为教学资源，实现"小课堂与社会大课堂相结合"。

**适配操作**：
- 案例须明确说明对应的现实场景（如哪个展馆、哪个基地、哪类社会实践）
- 设计"课前-课中-课后"三段联动任务：课前任务（现实参访/资料收集）→ 课中案例分析 → 课后反思报告
- 可设计"田野调研型"任务将案例延伸为实践活动

## 六、完整案例示范（三类素材各一例）

### 示范案例一：科学类素材（正面，理工科）

> **素材**：中国"天问一号"火星探测任务

---

**案例信息卡摘要**

| 字段 | 内容 |
|------|------|
| 案例标题 | 从地球到火星：天问一号与中国深空探测的战略布局 |
| 适用课程 | 轨道力学 / 航天工程导论（大三） |
| 专业知识锚点 | 行星际转移轨道（霍曼转移）、进入-下降-着陆（EDL）技术 |
| 主打思政主题 | 科学家精神（家国情怀 + 道德修养） |
| 模板类型 | 模板B（现象-本质-价值型） |
| 建议时长 | 10-15分钟 |

---

**【现象层】**

2021年5月15日，中国"天问一号"探测器携带"祝融号"火星车，成功在乌托邦平原南部着陆。中国由此成为继美苏之后第三个成功实现火星软着陆的国家，且是第一个首次探火便实现"绕、落、巡"三大目标的国家。此前，人类已进行过40余次火星探测，成功率约50%。

**【本质层】**（专业知识讲授）

行星际转移依赖"霍曼转移轨道"，利用地球与火星的会合周期（约26个月）选择最经济的发射窗口……（此处正常讲授专业内容）

*追问切入点*：中国为什么能在第一次就成功完成三大目标？"七分钟恐怖"的EDL过程是怎么攻克的？这背后是哪些团队，他们在什么条件下工作？

**【价值层】**

| 升华维度 | 具体内容 |
|----------|----------|
| 工程师的精神品质 | 攻关团队平均年龄33岁，数千名工程师历时15年攻克EDL技术，体现了坐冷板凳、甘于奉献的科学家精神 |
| 国家战略与个人选择 | 深空探测是国家重大科技工程，参与其中意味着长期保密、放弃更多短期机会，是一种主动的国家担当选择 |
| 制度支撑 | 举国体制在大型系统工程中的优势：协调数百家单位联合攻关，无法依赖纯市场机制实现 |

**【讨论问题】**

- *专业层*：霍曼转移轨道与直接飞行轨道相比有什么优劣？本次任务为什么选择它？
- *思政层*：如果你是2006年入职深空探测团队的年轻工程师，面对15年的漫长攻关和严格保密要求，你认为是什么在支撑这些人坚持下来的？

---

| 主打思政主题 | 法治意识 + 道德修养（技术伦理、劳动权益） |
| 辅助思政主题 | 政治认同（制度回应能力） |
| 素材性质 | 中性素材（批评性视角 → 引导至制度改进） |
| 模板类型 | 模板D（问题反思型） |

---

**【事件客观呈现】**

2020年9月，一篇题为《外卖骑手，困在系统里》的报道引发广泛讨论。报道揭示：平台算法为追求配送效率，不断压缩骑手的配送时间，导致骑手为赶时间违规抢灯、逆行，交通事故频发。三年间，上海某区外卖骑手交通事故伤亡人数呈上升趋势。

**【多元视角还原】**

- 骑手方：超时即被扣款，抢灯是"不得不"的选择，而非主动违规
- 平台方：算法设计以用户体验数据优化为导向，安全成本未被充分内化
- 消费者方：期待按时送达的需求无可厚非，但对系统压力知之甚少
- 监管方：传统劳动保护法律难以覆盖"灵活就业"群体

**【原因分层分析】**

| 原因类型 | 具体分析 |
|----------|----------|
| 技术设计 | 算法只优化效率指标，未将安全成本纳入目标函数 |
| 制度缺口 | 平台经济下的"个体工商户"定性使骑手缺乏劳动保护 |
| 市场压力 | 平台竞争导致"内卷式"时间压缩，个别平台改善动力不足 |

**【制度回应过程】**（关键）

报道引发关注后，国家市场监管总局、人社部等多部门陆续出台针对平台经济的专项规范（2021-2022年），要求平台为骑手提供安全保障、合理设置配送时限。多地开始探索"新业态劳动者"保障制度。部分平台增加了骑手保险覆盖和申诉机制。这一回应过程本身展示了中国制度的自我纠错能力。

**【思政价值点提炼】**

1. **技术伦理**：技术本身是中性的，但技术设计背后有价值选择——谁的利益被算法优化了？这引导学生思考"科技向善"的现实含义。
2. **劳动权益保护**：数字经济的发展不能以侵蚀劳动者权益为代价，劳动尊严是社会主义核心价值的基础。
3. **制度韧性**：中国治理体系能够对批评性舆论做出政策回应，这是制度优越性的体现之一（非对批评的回避，而是对问题的正视）。

**【讨论问题】**

- *理解层*：骑手为什么"不得不"违章？这个"不得不"背后是谁的责任？
- *分析层*：如果你是平台工程师，如何重新设计算法才能兼顾效率与安全？
- *反思层*：这件事说明技术进步在什么条件下才是真正的社会进步？
- *行动层*：作为消费者，你的哪些行为选择会影响骑手的工作处境？

---

| 建议时长 | 15-20分钟 |

**【情境还原】**

1934年10月，红军第五次反"围剿"失败，被迫开始战略转移，即后来的长征。转移初期，按照共产国际顾问李德的指挥，部队遭受重大损失：出发时8万余人，经过湘江战役后仅剩约3万人。在生死存亡的关头，党内一批领导人开始公开质疑既有路线的错误。1935年1月，在遵义召开政治局扩大会议。

**【冲突与抉择】**

会议上，博古做了关于第五次反"围剿"的总结报告，为错误路线做辩护；张闻天、毛泽东、王稼祥等人则提出批评意见。这一局面的特殊性在于：批评者是在质疑经共产国际认可的路线，在当时的国际共产主义运动背景下，这意味着极大的政治风险。但多数与会者选择了从实际出发而非服从权威。

**【行动与结果】**

会议通过决议，批评了博古、李德的军事指挥错误，确立了毛泽东在党中央的领导核心地位。长征此后转危为安，最终胜利到达陕北。遵义会议因此被称为"党的历史上生死攸关的转折点"。

**【思政映射】**

1. **实事求是的思想路线**：面对失败敢于承认错误、敢于纠正路线，而非为维护权威辩护，这是马克思主义实事求是精神的核心体现。
2. **党的自我革命能力**：遵义会议展示了中国共产党在极端困难条件下进行自我批评与自我纠正的能力，这一能力贯穿此后党的历史，也是理解当代党的建设的重要维度。
3. **历史合力与个人选择**：历史关键节点上，个人的正确选择改变了历史走向——思考"历史是由人民创造的"这一命题的具体含义。

**【讨论问题】**

- *事实层*：遵义会议前后党的军事指挥思路发生了什么变化？
- *分析层*：在当时的政治条件下，批评共产国际顾问的决策需要什么条件（政治勇气、组织条件、思想基础）？
- *价值层*：从遵义会议中，你如何理解"党的自我革命"这个概念？它在今天有什么现实意义？
