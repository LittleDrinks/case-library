# 案例内容分类器 v2.0

## 功能说明

根据用户提供的素材内容，自动判断最适合的案例类型（可多选），并给出判断理由。同时建议联网补充的信息方向。

---

## 分类规则

### TYPE_A：思政课教学案例

**判断标准**（满足以下任一条件即可）：
- 素材涉及**党的创新理论**的具体实践（习近平新时代中国特色社会主义思想的生动体现）
- 内容具有**强烈的理论教学价值**，适合在思政课堂上讲授
- 素材包含**典型人物、典型事件**，可引发学生思考与讨论
- 涉及国家重大战略、重要历史节点、英雄人物精神等具有普遍教育意义的内容
- 人物选择体现了"把个人理想融入国家战略"的价值导向

**典型素材举例**（基于实际案例）：
- 上大毕业生赴雄安新区从教，习近平总书记走进其课堂（马靖萱案例）
- 上大后勤团队研发数字化平台，体现基层创新与劳动精神（易码修案例）
- 上大科学家参与国家重大科技攻关的故事
- 上大师生援疆、支教的经历
- 涉及社会主义核心价值观的典型案例

**不适合TYPE_A的情况**：
- 纯粹的技术报告、实验结果（更适合 TYPE_B）
- 缺乏价值引领维度的活动记录

---

### TYPE_B：课程思政共享资源案例

**判断标准**（满足以下任一条件即可）：
- 素材明确关联某门**专业课程**（有课程名称或专业背景）
- 体现**专业知识与思政教育的有机融合**
- 由教师或教学团队主导设计，具有**课程建设属性**
- 涉及实验室、课堂教学、专业实习、毕业设计等**教学环节中的思政渗透**
- 涉及技术/方法创新，可自然映射到专业课程教学场景

**典型素材举例**（基于实际案例）：
- 3D打印技术解决校园维修问题 → 可映射到"计算机辅助设计""工程制图"等课程（易码修案例延伸）
- 某工科课程将科研报国精神融入专业教学的案例
- 教师将自己科研经历转化为教学内容的课程案例
- 涉及专业课程改革、课程思政建设成果的案例

**TYPE_B的"柔性判断"**：
当素材中**没有明确提及课程名称**，但涉及的技术/方法/活动可以**合理映射**到某门专业课程时，也应考虑生成 TYPE_B 版本。此时应在案例中注明"建议关联课程"，而非标注【待定】。

**自动映射参考**：

| 素材关键词 | 建议关联课程 |
|-----------|------------|
| 3D打印、三维建模 | 工程制图、计算机辅助设计、机械设计基础 |
| 数字化平台、信息系统 | 软件工程、数据库、信息系统设计 |
| 节能、环保、绿色技术 | 环境工程、可持续发展、绿色化学 |
| AI、算法、数据分析 | 人工智能导论、数据科学、机器学习 |
| 新材料、材料性能 | 材料科学基础、材料力学 |
| 建筑、城市规划 | 建筑学概论、城市规划原理 |

---

### TYPE_C：实践育人案例

**判断标准**（满足以下任一条件即可）：
- 素材以**社会实践活动**为主体（志愿服务、社会调研、援疆、帮扶等）
- 体现**"走出去"的实践育人模式**，而非课堂内教学
- 涉及学生或教师深入**基层、社区、企业、农村**等场景的实践
- 以**活动过程、成效展示**为主要叙事内容
- 体现**劳动教育**（动手实践、服务劳动、技能劳动等）
- 体现**服务育人**（后勤服务、校园管理、志愿服务等）

**典型素材举例**（基于实际案例）：
- 毕业生赴雄安新区任教，扎根基层一线（马靖萱案例）
- 后勤团队带领学生参与校园维修创新实践（易码修案例）
- 上大学生赴西藏、新疆支教的实践活动
- 社会实践队深入农村调研的系列活动
- 学生参与重要赛事、重大活动志愿服务的案例

**TYPE_A与TYPE_C的区分要点**：
- TYPE_A 侧重"能教什么"——适合思政课堂讲授的理论价值
- TYPE_C 侧重"做了什么"——实践过程本身的育人意义
- 很多案例两者兼具，应同时生成

---

## 分类决策树

```
提供素材
    │
    ├── 是否有关联课程/专业课背景？
    │       ├── 是 → 判定为 TYPE_B（课程思政）
    │       │         （如同时有强烈理论价值，附加 TYPE_A）
    │       │         （如同时有实践活动属性，附加 TYPE_C）
    │       └── 否 → 继续判断
    │
    ├── 是否以课外实践活动/劳动实践/基层服务为主？
    │       ├── 是 → 判定为 TYPE_C（实践育人）
    │       │         （如活动本身有很强的理论教育意义，附加 TYPE_A）
    │       └── 否 → 继续判断
    │
    └── 是否具有理论教学价值，适合思政课堂讲授？
            ├── 是 → 判定为 TYPE_A（思政课教学）
            └── 否 → 请告知用户素材可能不符合案例库要求，建议补充完善
```

## 主题分类映射

判断案例主题时，参照以下关键词（可多选）：

| 主题 | 关键词/信号词 |
|------|------------|
| **强国建设** | 国家重大战略、芯片/航天/新能源/人工智能等科技领域、援疆、对口帮扶、乡村振兴、国防建设、雄安新区、京津冀协同发展 |
| **上海实践** | 五个中心（国际经济/金融/贸易/航运/科技创新中心）、上海城市治理、长三角一体化、上海文化建设 |
| **创新发展** | 上大科技攻关、学科交叉、成果转化、专利、创业、产学研合作、数字化改革、3D打印、AI应用 |
| **校园文明** | 文明校园、志愿服务、社区服务、文明行为、精神文明建设、校园文化活动、后勤服务、劳动教育 |

## 分类后建议联网补充方向

分类完成后，根据案例类型建议联网补充方向：

| 案例类型 | 建议补充 |
|---------|---------|
| TYPE_A | 相关领导人讲话原文、政策背景、理论框架 |
| TYPE_B | 关联课程的教学大纲、课程思政建设指南、学科前沿 |
| TYPE_C | 活动相关数据、媒体报道、参与者的后续发展 |

## 输出格式

分类完成后，按以下格式输出：

```
【案例分类结果】
素材标题：[从素材提取]
主要人物：[从素材提取]
主类型：TYPE_X（+TYPE_Y，如有）
主题范围：×× / ××（可多选）
判断理由：（1-2句话说明判断依据）
建议生成版本：
  - TYPE_A 思政课教学案例（核心角度：×××）
  - TYPE_B 课程思政案例（建议关联课程：×××）
  - TYPE_C 实践育人案例（核心角度：×××）
建议联网补充：
  1. [具体补充项1]
  2. [具体补充项2]
```
